# Volume Knob

Project remix of the [Prusa oversized volume knob](https://blog.prusa3d.com/3d-print-an-oversized-media-control-volume-knob-arduino-basics_30184/).

<img src="./pictures/volume_knob.png" alt="volume_knob.png" width="300"/>
<img src="./pictures/volume_knob_exploded.png" alt="volume_knob_exploded.png" width="300"/>

## Bill of material

- Arduino Pro Micro (USB-C or Micro-USB)
  </br><img src="./pictures/arduino_pro_micro.png" alt="arduino_pro_micro.png" width="200"/>
- USB cable
- Rotary encoder with button (KY-040)
  </br><img src="./pictures/rotary_encoder.png" alt="rotary_encoder.png" width="200"/>

- 5 wires (25mm long)

## Instructions

### Wiring

Remove headers from both Arduino and rotary encoder and solder wires directly to contacts on PCB.

### 3D printing settings

|              |            |
| ------------ | ---------- |
| Printer      | Ender 3 S1 |
| Layer height | 0.2mm      |
